# Changelog

## Q1 2026 (Q126) — First Release

Initial public release of Open Access Kit.

### What's included

- S, M, and L tier distributions
- Tor Browser (all platforms), Orbot, OnionShare, Tails OS
- Offline guides: Tor Browser Manual, Security in a Box, Privacy Guides,
  Digital First Aid Kit, Tails Documentation, SaferJourno
- OAK guides: getting started, release manifest, resources, onion site directory
- ZIM offline reference content (P0 + P1 in M tier; P0 + P1 + P2 in L tier)
- GPG-verified downloads; OAK release key signature on final image
