A curated directory of privacy tools, digital security guides, and circumvention resources. Items marked [bundled] are available offline on this removable media.

Privacy & Anonymity Tools

Resource License Notes
Tor Browser MPL-2.0 [bundled] Anonymizing browser. The foundation of private browsing. Docs
Tails OS GPL-3.0 [bundled, M and L] Live OS. Leaves no trace on host computers. Docs
OnionShare GPL-3.0 [bundled, M and L] Share files, host websites, and chat over Tor. Guide
Orbot GPL-3.0 [bundled] Tor for Android apps system-wide. Guide
