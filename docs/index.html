Open Access Kit (OAK)

A portable, offline-first collection of privacy tools and educational content for people living under censorship or surveillance. Everything you need fits on removable media.

Latest Release: Q126

  • OAK-Q126-L.zip · max 32 GB — Download · SHA256 · Signature

What's on this drive?

OAK comes in three sizes to fit the removable media you have:

  • S (max 4 GB) — Tor Browser (all platforms), Orbot, offline security guides.
  • M (max 16 GB) — Everything in S, plus Tails OS, OnionShare, Kiwix with offline Wikipedia, medical references, and educational content.
  • L (max 32 GB) — Everything in M, plus full Wikipedia mini, Wikibooks, and expanded reference libraries.

Full manifest with versions, sizes, and checksums

Getting started

  1. Open the software/tor-browser/ folder on this removable media.
  2. Install Tor Browser for your operating system.
  3. Launch Tor Browser and click Connect.
  4. You're connected. Browse .onion sites using the Onion Sites directory, or read the guides in this folder.
